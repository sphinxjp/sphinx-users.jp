========================================
FreeBSDへのTeX liveインストール
========================================

FreeBSDでは、/usr/ports/print/texlive-full をビルドすると TeX Live がインストールされます。

.. code-block:: bash

   % cd /usr/ports/print/texlive-full
   % sudo make install


.. note::

  2014年1月時点で、TeX Live 2012 がインストールされました。
