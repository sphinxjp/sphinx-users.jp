========================================
FreeBSDへのTeXliveインストール
========================================

FreeBSDでは、/usr/ports/print/texlive-full をビルドすると TexLiveがインストールされます。

.. code-block:: bash

   % cd /usr/ports/print/texlive-full
   % sudo make install


.. note::

  2014年1月時点で、TexLive2012がインストールされました。
